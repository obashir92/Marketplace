Full stack Django-project showcase
(Fully functional Marketplace)

Front-end: Html5 / Tailwind(Css)
Back-end: python django framework
Version control: Github

Features

- Add/remove/edit products (database management)
- Product details eg. Price, Description, status..
- User creation & Authentication
- Dashboard
- Search items
- Buyer/Seller communication channel
